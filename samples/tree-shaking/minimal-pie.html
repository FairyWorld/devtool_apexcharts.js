Minimal Pie Chart

Imports: core + pie + tooltip. Legend is intentionally omitted to show it's a separate optional feature.

// Legend is NOT imported — the chart renders without it
import ApexCharts from '../../src/entries/core.js'
import '../../src/entries/pie.js' // pie, donut, polarArea
// import '../../src/features/legend.js' ← not loaded
The legend is not registered, so it will not appear even if legend.show is not explicitly set to false. This is the tree-shaking contract: unregistered features are silently skipped.
