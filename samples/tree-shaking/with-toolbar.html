Line + Toolbar & Zoom

Adds toolbar + ZoomPanSelection on top of the minimal line chart. These are always registered together — zoom/pan controls require both modules.

import ApexCharts from '../../src/entries/core.js'
import '../../src/entries/line.js'
import '../../src/features/legend.js'
import '../../src/features/toolbar.js' // registers Toolbar + ZoomPanSelection
The toolbar feature registers both Toolbar and ZoomPanSelection in one import. Try the zoom-in, zoom-out, and pan buttons in the toolbar above the chart.
