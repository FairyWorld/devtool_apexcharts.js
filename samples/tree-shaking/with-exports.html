Bar + Export

The exports feature adds PNG/SVG/CSV download to the toolbar. Without this import, the download menu items in the toolbar are absent.

import ApexCharts from '../../src/entries/core.js'
import '../../src/entries/bar.js'
import '../../src/features/legend.js'
import '../../src/features/toolbar.js' // zoom/pan UI
import '../../src/features/exports.js' // PNG / SVG / CSV download
Click the hamburger menu (⋮) in the top-right of the chart to access PNG, SVG, and CSV export options. These only appear because exports.js is imported.
